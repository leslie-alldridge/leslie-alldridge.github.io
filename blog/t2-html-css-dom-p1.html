HTML, CSS & DOM

  • Can you think of an analogy to describe HTML, CSS and the DOM? How would you describe it to your non-technical friend?

    • If we were to think of HTML, CSS and the DOM in terms of a car. The HTML would be the shell of the car, seats, engine and other parts the car need to run. The CSS would be the color of the shell, color of the seats and other style elements the car has. The DOM is an action, so when the key is turned, the engine starts. The window going down is the DOM as it's manipulating the window to become smaller.
